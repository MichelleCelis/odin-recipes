--Overview--
For The Odin Project: this project is a recipe website, designed and developed using HTML and CSS.
It provides a platform for users to browse and choose from a limited number of recipes.
Additional sections were added: an 'About' section to welcome the user, and a 'Contact Us' section - currently not functioning.

--Skills Demonstrated--
> HTML: structuring web content in a semantic manner.
> CSS: styling and layout to provide an intuitive and visually appealing user experience.

--Project Structure--
> index.html: the main landing page.
> styles.css: stylesheet for landing page.
> recipes/: directory containing invidiual recipe html and css files.

--Self-Reflection--
Improvement:
> HTML; faster/easier to structure a webpage.
> Flexbox; gained understanding of aligning elements.
> CSS; faster/easier to style and layout a page.
> Animations; used a rudimentary animation to fade-in landing page elements upon loading.
> Search Bar; learnt how to create a search bar that links to Google when used.
